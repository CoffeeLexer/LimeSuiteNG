Device support list
================================

* USB interface:

    - `LimeSDR USB`_
    - `LimeSDR Mini v1`_
    - `LimeSDR Mini v2`_

* PCIe interface (currently PCIe driver available only for Linux):

    - `LimeSDR XTRX`_
    - `LimeSDR X3`_
    - LimeMM X8

* To be done:
    - `LimeRFE`_

.. _LimeSDR XTRX: https://limesdr-xtrx.myriadrf.org/
